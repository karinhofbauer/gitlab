Git - lab

The intention with this lab is to give an understanding of what Git is and how it works. It will show the benefits of using Git as your versioning tool and hopefully make you comfortable enough to start using it in your new and current projects. The lab consists of 4 different parts:

  1. Basics

    Shows the basic commands of git. Adding values to the config, adding files, committing files, moving files and deleting files.

  2. Branching

    Create branches and learn how to switch in between them.

  3. Rebasing and merging

    Merge branches or rebase and push. Delete branches when done with them.
